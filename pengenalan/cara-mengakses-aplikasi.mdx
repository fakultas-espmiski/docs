Untuk menggunakan aplikasi **eSPMI**, pengguna dapat mengakses sistem melalui browser pada perangkat yang terhubung dengan jaringan internet. Pengguna harus memiliki **akun yang telah terdaftar** pada sistem untuk dapat masuk ke dalam aplikasi.

### Langkah-langkah Mengakses Aplikasi

<AccordionGroup>
  <Accordion title="Buka browser pada perangkat Anda">
    Anda dapat mengakses browser seperti Google Chrome, Mozilla Firefox, atau browser lainnya, kemudian masukkan **alamat URL aplikasi eSPMI** pada kolom address bar.

    <Frame>
      ![Fakultas Link](/images/Fakultas.link.png)
    </Frame>
  </Accordion>

  <Accordion title="Masukkan Username dan Password pada kolom yang tersedia.">
    Silakan akses halaman login e-SPMI dan masukkan akun Anda untuk mulai mengelola data penjaminan mutu.

    <Frame>
      ![Fak Login Username Akun](/images/fak.login.username-akun.png)
    </Frame>
  </Accordion>

  <Accordion title="Isikan Captcha">
    Isikan captcha sesuai dengan angka penjumlahan yang ditampilkan. Captcha akan refresh setiap 30 detik.

    <Frame>
      ![Fak Login Captcha](/images/fak.login.captcha.png)
    </Frame>
  </Accordion>

  <Accordion title="Tekan Tombol Login">
    Anda akan diarahkan pada halaman Dashboard apabila berhasil dan tetap di halaman Login apabila gagal melakukan login.

    <Frame>
      ![Fak Login Login](/images/fak.login.login.png)
    </Frame>
  </Accordion>
</AccordionGroup>
