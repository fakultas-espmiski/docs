---
title: "Cara Ubah Password"
icon: "circle-small"
---

Fitur **Ubah Password** digunakan untuk membantu pengguna mengganti kata sandi akun secara mandiri melalui aplikasi **eSPMI**. Penggantian password dapat dilakukan secara berkala untuk menjaga keamanan akun pengguna saat mengakses sistem.

<Note>
  Gunakan password yang mudah diingat namun tetap aman. Simpan informasi password dengan baik dan jangan membagikannya kepada pihak lain.
</Note>

### Langkah-langkah Mengubah Password

<Steps>
  <Step title="Login ke aplikasi eSPMI menggunakan akun Anda.">
    Anda dapat login menggunakan username dan password Anda.
  </Step>
  <Step title="Klik ikon profil / nama pengguna">
    Anda dapat melakukan klik pada ikon profil/nama pengguna yang terletak di pojok kanan atas.

    <Frame>
      ![Spmi Login Klik Profil 1](/images/spmi-login-klik-profil-1.png)
    </Frame>
  </Step>
  <Step title="Pilih menu Profil > Ubah Password">
    Untuk mengubah password, Anda dapat melakukan klik pada tombol Ubah Password.

    <Frame>
      ![Spmi Profil](/images/spmi-profil.png)
    </Frame>

    <Frame>
      ![Login Profil Ubah Pass](/images/login-profil-ubah-pass.png)
    </Frame>
  </Step>
  <Step title="Isi form Ubah Password">
    Anda dapat memasukkan **Password Baru** sesuai dengan password yang ingin digunakan. Kemudian, ketik ulang **Konfirmasi Password Baru** untuk memastikan password yang dimasukkan sudah sesuai.

    <Frame>
      ![Image](/images/image.png)
    </Frame>
  </Step>
  <Step title="Klik tombol Simpan">
    Tombol **Simpan** untuk menyimpan perubahan. Apabila proses berhasil, maka password akun Anda akan diperbarui dan dapat digunakan untuk login pada sesi berikutnya.

    <Frame>
      ![Spmi Profil Ubah Pass Berhasil](/images/spmi-profil-ubah-pass-berhasil.png)
    </Frame>
  </Step>
</Steps>
