---
title: "Cara Ubah Password"
description: "Panduan untuk mengubah kata sandi akun eSPMI Anda."
---

Untuk menjaga keamanan akun, disarankan untuk mengubah password secara berkala:
1. Masuk ke menu **Profil** atau **Pengaturan Akun**.
2. Pilih opsi **Ubah Password**.
3. Masukkan password lama dan password baru Anda.
4. Klik **Simpan**.
